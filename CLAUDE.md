# kyouen_dart

Dart パッケージ。4点の共円判定ライブラリ。

## バージョンアップ手順

ユーザーから「バージョンを X.Y.Z にしてください」と依頼された場合、以下を**すべて自動で**実行する。

1. **`pubspec.yaml`** の `version:` フィールドを新バージョンに更新する
2. **`CHANGELOG.md`** の先頭に新バージョンのセクションを追記する
   - 形式: `## X.Y.Z` の見出しに続いて、今回の変更内容を箇条書きで記述する
   - 変更内容はセッションの作業履歴から自動的に要約する
3. 両ファイルをまとめて git commit する（メッセージ例: `Bump version to X.Y.Z`）
4. 現在のブランチに git push する

ユーザーが明示的に確認を求めていない限り、途中で確認せず一気に完了させる。
